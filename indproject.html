About this Project

Artible is a design-led project done in collaboration with Coursera's Google UX Design Certification program. Artible is a mobile application that provides audio tours for museums, designed to enhance the museum experience for visitors of all ages and interests. My goal was to design an easy-to-use mobile application that museum visitors and people with visual impairments can use to learn more about art, creating a more enjoyable and educational experience for everyone.
